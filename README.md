# 📘 Exercícios e Estudos em Python

Repositório dedicado ao registro da minha jornada de aprendizado em **Python**.  
Aqui estão organizados os exercícios e exemplos desenvolvidos durante o curso que estou realizando, começando desde os conceitos mais básicos da linguagem até aplicações simples em terminal.

O objetivo deste repositório é **documentar minha evolução prática**, reforçando conceitos fundamentais de programação e lógica.

---

## 📚 Conteúdos Estudados

Entre os principais tópicos abordados até o momento estão:

- print() e saída de dados  
- Variáveis e tipos de dados  
- Entrada de dados com input()  
- Conversão de tipos (int, float, str)  
- Operadores matemáticos e lógicos  
- Estruturas condicionais (if, elif, else)  
- Manipulação de strings  
- Validação de dados  
- Tratamento de erros com try/except  
- Estruturas de repetição (while)  
- Lógica de programação  
- Construção de pequenos programas em terminal  

---

## 🧠 Exercícios Desenvolvidos

Alguns exemplos de exercícios presentes no repositório:

- Verificação de número par ou ímpar
- Identificação de tipo de entrada do usuário
- Programa de saudação baseado no horário
- Análise de tamanho de strings
- Calculadora simples em terminal utilizando while
